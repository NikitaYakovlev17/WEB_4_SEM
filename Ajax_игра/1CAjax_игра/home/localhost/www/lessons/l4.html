Фух, теперь можно вдохнуть полной грудью. Теперь необходимо открыть двери из командного блока. Дверьми управляет файл door.php в каталоге systems. Необходимо передать в него параметр door со значением 1 и status - open. Тип запроса - GET После того, как переменная запроса создана, необходимо вызвать метод open, который принимает несколько аргументов:
  1. Тип запроса - "GET" или "POST"
  2. URL файла - к примеру "folder/file.php"
  3. Асинхронно ли выполнить запрос? Если да, то true, иначе - false
xhttp.open("ТИП ЗАПРОСА", "URL", Асинхронно?); После этого необходимо вызвать метод send чтобы отправить запрос.
Отредактируйте параметры в методе open так, чтобы сформировать необходимый запрос по условию выше.
